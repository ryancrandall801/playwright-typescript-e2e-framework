# playwright-typescript-e2e-framework
Playwright typescript e2e framework.

## Running Tests

### Locally
npm test

### With Docker
npm run docker:build
npm run docker:test

### CI
Tests run automatically on pull requests and pushes to main via GitHub Actions.
